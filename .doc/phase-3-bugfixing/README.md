## 📦 Commands (Planned)

```bash
# Detect and fix bugs
docai fixbug [path]

# Show specific bug types
docai fixbug --type security [path]
docai fixbug --type performance [path]

# Explain only (no fixes)
docai fixbug --explain-only [path]

# Auto-fix critical bugs
docai fixbug --auto-critical [path]
```
